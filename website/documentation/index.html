Step - Responsive Personal Portfolio Template

documentation

  • Created: 01/12/2019
  • Latest update: 01/12/2019
  • By: ThemeMix

Step - is a Responsive Personal Portfolio Template suitabe for Resume or Portfolio websites designed for creative designers, developers, freelancers or pretty much any profession. This Template is 100% responsive, well organized and very easy to customize..

It's available in Light and Dark Versions. Step is better way to present your modern Portfolio/ Resume.

turn off Preloader

Open the index.html file using any text editor then look for the following code and remove them. 

<!-- ========== Start Loading ========== -->

<div class="preloader">
    <div class="loading-inner">
    </div>
</div>

<!-- ========== End Loading ========== -->

Save the file and refresh your browser.

Home

To change home background image put your new image in 'images' folder (recommended size 1920px * 1080px), The name of image should be 'home' and image extension should be 'jpg'.
Then open the css/main.css file using any text editor then look for the following code and replace blue highlighted text with your new image's path, name and extension like this url('../images/home.jpg'). 

.home {
    background: url('https://via.placeholder.com/1920x1080') center center no-repeat fixed;
    -webkit-background-size: cover;
    -moz-background-size: cover;
    -o-background-size: cover;
    background-size: cover;
    position: relative;
    height: 100vh;
    overflow: hidden;
}

To edit the text in 'Home' Open the index.html file using any text editor then look for the following code and replace yellow highlighted text.

<!-- ========== Start Home ========== -->

<section class="home" id="home">

            <div class="intro display-table">
